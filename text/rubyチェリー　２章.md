## return(返り値)について
rubyは基本returnは省略する。しかしメソッドの途中で脱出する場合に使う時がある。
```
def greeting(country)
    return 'countryを入力してください' if country.nil?

    if country == 'japan'
        'こんにちは'
    else
        'hello'
    end
end

p greeting(nil) #=> countryを入力してください'
p greeting('japan') #=> 'こんにちは'
```
# 掘り下げ
## %記法
%! ! や%Q! !で挟むと文字列にできる。!は終わりの文字によって[,{,<になったりする。
```
puts %q!He said "Hello"! #=> He said "Hello"
```
## ヒアドキュメント
文字列は改行できる。複数行の時はヒアドキュメントを使う。<<-,<<〜などの書き方もある。upcaseなどのメソッドも呼び出せる。
```
a = <<TEXT
  これはヒアドキュメントです。
　これはヒアドキュメントです。
TEXT

puts a #=>   これはヒアドキュメントです。
```
## ０文字でも文字列
""といった文字が入っていなくても文字列と判断される。
