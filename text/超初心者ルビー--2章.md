## 2-5
- irbは一行ずつコマンドを実行できる。exitで終了
### プログラムを一時停止してirbを使う
- biding.irbを入れると、そこまでプログラムを実行してくれる

```
a = 1
binding.irb
puts a

$ ruby birb.rb

From: birb.rb @ line 2 :

    1: a = 1
 => 2: binding.irb
    3: puts a
```
既にある変数に代入する事もできる
## 2-6
### デバック
プログラムが動いていない部分をバグ、バグを調べて修正することをデバッグという。
### pメソッド
pメソッドの後ろに書いた変数やオブジェクトを画面に表示する。Putsメソッドと同じ機能っぽい

```
a = 3 / 2
p a
b = a * 2
puts b

$ ruby p2.rb 
1
2
```
