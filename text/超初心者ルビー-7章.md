## メソッド
メソッドは処理部品に名前をつけたもの。
## メソッドにオブジェクトを渡す
メソッドに引数をセットしておいて、メソッドを呼び出すときに引数にオブジェクトを指定することで、メソッドにオブジェクトを渡せる。
## return
returnを書くとそこで処理が終わる。
## 戻り値
return 〜　の〜に書いたオブジェクトが戻り値になる。
## 引数のデフォルト値
def order(item = coffie)の感じでデフォルト値を設定できる
## キーワード引数
def order(size:,item:)のように引数を設定できる。呼び出すときに、order(size:"ショート",item:"カフェラテ")のように順番関係なく引数を渡せる。
メソッド内では引数のコロンは消してかく。
## ローカル変数
def endの間でしか使えない変数。そのメソッドの外で呼び出してもエラーが出る。
